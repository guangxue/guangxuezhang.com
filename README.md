# [guangxuezhang.com](https://guangxuezhang.com)

A personal website written in Next.js with TypeScript and Tailwindcss. Built and maintained by [@guangxuezhang](https://guangxuezhang.com)

## Tech stack

[Next.js](https://nextjs.org/)

[next-mdx-remote](https://github.com/hashicorp/next-mdx-remote)

[next-auth](https://authjs.dev/)

[Prisma](https://www.prisma.io/)

## Backend features

### Admin Panel

1. TextEditor with features
   - Vim Keymaps: (**in beta version**)
   - Markdown syntax highlighting: (**in beta version**)
2. Save post to database using Prisam
3. Next-Auth role-base login using credentials

### User Panel

1. Nex-Auth login with Providers: (Google & Github)
2. comment system (**not yet**)
